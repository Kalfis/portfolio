SMASHED

A reddit-like posting app built entirely in Ruby on Rails. Made with the theme of the user being "smashed" and was based on an idea I had when talking to people in my General Assembly cohort. We thought "wouldn't it be funny if someone made an app for people to drunk post?" And so SMASHED was born.

Bar Finder

Continuing with the drinking theme of SMASHED, Bar Finder is a simple javascript app that uses google's geolocation API to find local bars near the user that are open. It finds the user and creates a map, then by pressing the simple "Bar for Bars" button, locates all the nearby drinking establishments
